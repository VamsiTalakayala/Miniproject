HTML Grouping with Div and Span

Using the <div> Tag

The <div> tag is used to group block-level elements together. It is often used to create sections of content that can be styled or positioned using CSS.

Example:

This is a paragraph inside a <div> element.

Another paragraph inside the same <div> element.

<div class="box">
    <p>This is a paragraph inside a <div> element.</p>
    <p>Another paragraph inside the same <div> element.</p>
</div>

Using the <span> Tag

The <span> tag is used to group inline elements or text for styling purposes. It does not introduce any block-level separation, so itâ€™s typically used for styling parts of a text.

Example:

This is a text with a highlighted word using the <span> tag.

<p>This is a text with a <span class="highlight">highlighted</span> word using the <span> tag.</p>

Combining <div> and <span>

The <div> and <span> tags can be used together to structure and style content effectively.

This is a paragraph with a highlighted word inside a <div> element.

<div class="box">
    <p>This is a paragraph with a <span class="highlight">highlighted</span> word inside a <div> element.</p>
</div>
