HTML Basic Formatting Tags

HTML Basic Tags

HTML basic tags are the essential building blocks for creating a webpage. Here are some commonly used tags:

  • <h1> to <h6>: Heading tags, with <h1> being the highest level and <h6> the lowest.
  • <p>: Paragraph tag, used for defining blocks of text.
  • <br>: Line break tag, used to insert a line break within text.
  • <hr>: Horizontal rule tag, used to insert a horizontal line.
  • <a>: Anchor tag, used to create hyperlinks.
  • <img>: Image tag, used to embed images in a webpage.

Example:

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<a href="https://www.example.com">This is a link</a>

HTML Formatting Tags

HTML formatting tags are used to emphasize text or change its appearance. Some of the commonly used formatting tags are:

  • <b>: Makes the text bold.
  • <strong>: Emphasizes the text with strong importance (typically displayed in bold).
  • <i>: Italicizes the text.
  • <em>: Emphasizes the text with a stress (typically displayed in italics).
  • <u>: Underlines the text.
  • <s>: Strikes through the text.
  • <sup>: Makes text appear as superscript.
  • <sub>: Makes text appear as subscript.
  • <mark>: Highlights the text.
  • <small>: Reduces the text size.
  • <blockquote>: For quoting blocks of text.

Example:

<p>This is <b>bold</b>, <i>italic</i>, and <u>underlined</u> text.</p>

HTML Color Coding

HTML allows you to set text and background colors using the style attribute, CSS classes, or inline styles. You can specify colors by name, hex code, RGB values, or HSL values.

Here are a few examples:

  • color: red;: Sets the text color to red.
  • color: #ff0000;: Sets the text color using a hex code.
  • color: rgb(255, 0, 0);: Sets the text color using RGB values.
  • background-color: yellow;: Sets the background color to yellow.

Example:

<p style="color: blue;">This text is blue.</p>
<p style="color: #ff6347;">This text is tomato color.</p>
<p style="background-color: yellow;">This text has a yellow background.</p>

In the example below, the class .colored-text is used to style text:

<p class="colored-text">This text is styled with a CSS class.</p>

This text is styled with a CSS class.
